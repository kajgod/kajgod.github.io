# Kajgod tech blog

A blog about tech & tech philosophy

## Installation

Requires Node.js version 20+

```bash
  cd kajgod-blog
  npm install
```

### Start dev

```
  npm run dev
```

### First build

There aren't old files for removal

```
   npm run newbuild
```

### Normal build

```
  npm run build
```

### Build with bundle analyzer

```
  ANALYZE=true npm run build
```

## Tech Stack

**Client:** Next.js, React

**Build tools:** Node, (Go)
